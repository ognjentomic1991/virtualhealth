# virtualhealth
Program for graduating exam

This is a virtual health graduated exam.

I developed a JavaFX application by using JavaFx technology in IntelliJ IDEA.

Tech stack:

   - IntelliJ IDEA

   - Scene builder

   - Swing 



Requirements:

For building and running the application you need:


   - JDK 1.8 or newer

   - IntelliJ IDEA

   - Scene builder

   - Xampp localhost


Port:

   - Localhost: 8080
